Asynchronous Programming


						void synchronous() {
						    final request = SyncHttpClient.getUrl(Uri.http('10.0.2.2:5000', '/'));
						    final response = request.close();						
						    dialog(response.body);
						}
					

						void asynchronous() async {
						    final future = http.get('http://10.0.2.2:5000');
						    final response = await future;
						    dialog(response.body);
						}

Both look pretty much the same!
